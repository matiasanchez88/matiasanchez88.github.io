Configurá Supabase

Para que los datos sean visibles entre dispositivos, necesitás conectar tu proyecto de Supabase. Seguí los pasos:

  1. 1Entrá a supabase.com, creá una cuenta gratis y un proyecto nuevo.
  2. 2En tu proyecto Supabase andá a Settings → API y copiá la Project URL y la anon public key.
  3. 3En el editor SQL de Supabase ejecutá el script que hay en el archivo supabase-setup.sql (incluido en la descarga).
  4. 4Pegá tu URL y key en las primeras líneas del script de este index.html donde dice TU_SUPABASE_URL y TU_SUPABASE_KEY.
// Línea ~50 del index.html:
const SUPABASE_URL = 'https://xxxx.supabase.co'
const SUPABASE_KEY = 'eyJh...tu-anon-key'

Mientras tanto, la app funciona en modo local (los datos no se comparten entre dispositivos).

Política de Privacidad

1. Datos que recolectamos. Al registrarte, recolectamos tu dirección de correo electrónico. Los datos públicos de tu tienda son visibles para todos los visitantes.

2. Uso de los datos. Los datos personales se utilizan exclusivamente para autenticarte y asociarlos a tu tienda. No los vendemos ni compartimos con terceros.

3. Almacenamiento. Los datos se almacenan en servidores de Supabase (infraestructura en la nube segura) con encriptación en tránsito (HTTPS) y en reposo.

4. Ley 25.326. Podés solicitar el acceso, rectificación o eliminación de tus datos escribiendo a contacto@almayoreo.com.ar.

5. Cookies. Usamos cookies de sesión para mantener tu login activo. No utilizamos cookies de rastreo ni publicidad.
